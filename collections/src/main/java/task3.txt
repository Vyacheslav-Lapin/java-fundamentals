Set:
Основана на HashMap. Хранит объекты неупорядоченно, нельзя добавить пару одинаковых элементов.
Map: Пара ключ-значение. Хранит объекты неупорядоченно, при добавлении одинаковых ключей, старый будет перезатираться. Значения могут повторяться
List: Самая популярная коллекция. ArrayList - основан на массиве, Linked - на двусвязном списке.
Queue: FiFo - первый пришёл, первый ушёл.
