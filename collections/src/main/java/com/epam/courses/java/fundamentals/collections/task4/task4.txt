Set:
 -Основана на HashMap.
 -Хранение объектов - неупорядоченное.
 -Нельзя добавить одинаковые объекты.

List:
 -Интерфейс. Применяется для создания простых списков.
 -Реализация ArrayList - основана на массиве. При приближении к заполнению массива - размер увеличивается по формуле: "текущий размер" * 1.5 + 1
 -Реализация LinkedList - основана на списке.

Queue:
 -Интерфейс. Применяется для создания очереди по типу FIFO(Первый вошёл - первый вышел)
 -Реализация - ArrayDeque

Map:
 -Интерфейс. Предаставляет с собой отображение, где каждый элемент представляет пару ключ - значение.
 -Объекты хранятся неупорядоченно.
 -Может содержать повторяющиеся значения.
 -При добавлении ключа, который уже присутствует в коллекции - старый будет перезаписан.
